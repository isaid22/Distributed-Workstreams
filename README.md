# Distributed-Workstreams

This repo is for demonstration of using IntelliJ to compile and run Spark code locally. 

## Prerequisite

Below are steps to setup a Scala project in IntelliI IDE:

1. Open IntelliJ
2. Install Scala plugin: Settings → Plugins → Search "Scala" → Install
3. Create New Project → Scala → sbt
4. Write code, click green "Run" button
